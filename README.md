## 💻 Commands I Am Practicing

Some examples of Linux commands I am learning:

### Check current directory

```
pwd
```

### List files and directories

```
ls
ls -l
ls -a
```

### Change directory

```
cd foldername
cd ..
```

### Create directory

```
mkdir new_folder
```

### Remove file

```
rm filename
```
